# Loan-Default-Financial-Risk-Analysis-Power-BI

This project demonstrates a Power BI dashboard built using SQL Server and Power BI Dataflows as the data source.

Data Flow Architecture
SQL Server → Data storage
Power BI Dataflows → Data connection layer
Power BI Desktop → Data modeling, DAX calculations, and visualization

Key Features
Loan default analysis across demographics and financial attributes
Segmentation by credit score, income, employment type, and age group
YoY and YTD KPI tracking

Tools Used
SQL Server
Power BI Dataflows
Power BI Desktop
DAX

Insights
Higher defaults in low credit score and unemployed segments
Income and employment type strongly influence financial risk
